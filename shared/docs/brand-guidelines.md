# Keyxif Brand Guidelines

Keyxif Android and Keyxif Web share the same product name, icon direction, template names, and neutral keyboard-focused tone.

## Shared Rules

- Keep Android and Web independently buildable.
- Use `shared/schema` as the reference for portable build/export metadata.
- Update `shared/templates/template-specs.json` before introducing a new public template.
- Keep third-party vendor marks out of shared assets unless redistribution permission is clear.

## Logo Variant Rules

- Default logo: use when it remains readable on both light and dark card backgrounds.
- `_w` logo: use on dark backgrounds.
- `_b` logo: use on light backgrounds.
