# Shared Logo Assets

This folder is the source-of-truth location for logo originals that can be prepared for both Android and Web.

Naming convention:

- `logo_name.png`: default logo that works on mixed backgrounds
- `logo_name_w.png`: white/light logo for dark backgrounds
- `logo_name_b.png`: black/dark logo for light backgrounds

Android drawable resource names must stay lowercase with underscores. Web filenames should follow the same convention where possible.

Before adding third-party brand logos, confirm that the app has permission to redistribute them.
